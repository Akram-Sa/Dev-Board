Question-1: What are the different ways to select an element in the DOM?

Document Object Model (DOM) provides several methods to select and manipulate elements on a web page. Selecting elements is a fundamental step in dynamically updating content, handling events, and modifying styles.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML retrieves or sets an element's content, including HTML tags, while innerText retrieves only the visible text, ignoring hidden elements. textContent provides all text content, including hidden elements, without processing HTML tags, making it faster and more reliable.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single parent element handles events for multiple child elements instead of assigning event listeners to each child individually. This is possible because events in JavaScript bubble up from the target element to its ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is a phase in the event propagation process where an event starts from the target element and moves upward through its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement("tagName"), customize it with attributes or text, and add it using appendChild(), append(), prepend(), or insertBefore(). To remove an element, use parent.removeChild(childElement) or element.remove() to delete it directly from the DOM.
